Installing the Newf Repository

Current Repository Version: 2.0.

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://newf276.github.io/
  • Name it "Newf's Repo" without the quotes
  • Head to "My Addons"
  • Select install an addon from zipfile.
  • When it asks for the location, select "Newf's repo", and install repository.newf-2.0.zip.
  • Go back to My Addons and select "Install from repository"
  • Select the "Newf's Repo" menu item
  • Open and install what you like

Enjoy!
